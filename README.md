hon-provisu-extension
=====================

**Pro Visu Look**: Background, text color and font size change.

Install
-------

### extension

```bash
$ npm install
$ bower install
$ gulp build
```

### web service

```bash
$ npm install
$ bower install
$ gulp build-web
```

### toolbar

```bash
$ npm install
$ bower install
$ gulp build-toolbar
```

Run
---

### web

```bash
$ gulp server
```

A server will be launched on http://localhost:3000

### extension

from google Chrome:

* Go to Settings ==> Extensions
* Enable Developer mode
* load unpacked extension...
* select `/dist`

### toolbar

```bash
$ gulp serve-toolbar
```
